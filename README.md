Password Strength Analyzer

A professional Flask-based Password Strength Checker that analyzes password security in real time using entropy calculation, character diversity checks, and common password detection.

---

Features

- Real-time password strength analysis
- Entropy calculation
- Common password blacklist detection
- Password visibility toggle (show/hide)
- Secure password scoring system
- Dynamic progress bar
- Security improvement suggestions
- Password history logging
- Responsive modern UI
- Professional glassmorphism design
